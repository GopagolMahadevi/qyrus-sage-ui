About Normal Mode?

In Normal mode of testing your bot, we do not apply any type of text transformations while testing your bot via the paraphrases we have generated.
It will be more of a vanilla test. To test your bot functionality with spelling mistakes, grammatical mistakes, null sentence structures, contractions, etc. You can choose to opt for either of the two modes mentioned below based on your requirements.

About Human Error Mode?

In this mode we apply seven different transformation in total and more to come. These transformations will mimic the writing/typing quality of non-english users.

The following transformations are applied:

Spelling Mistake Transform
Word Swap Transform
Text Contractions
About Advance Mode?

In this mode we apply seven different transformation in total and more to come. These transformations will mimic the writing/typing quality of non-english users.

The following transformations are applied:

Spelling Mistake Transform
Word Swap Transform
Text Contractions
Null/No Sentence Structure
Using untrivial words
Similar word repetition
Similar word substitution
Start Paraphrases
Total Sentences
